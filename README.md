# Poster App

An extraordinary data poster application featuring a beautiful 2x5 grid layout with vibrant colors, gradients, and subtle patterns to visualize data in a highly aesthetic way.

## Features
- **2x5 Grid Layout**: A visually striking grid designed specifically to present 10 key data points.
- **Vibrant Color Scheme**: Carefully selected gradients and contrasting colors to make the poster pop.
- **Responsive Design**: Adapts gracefully to different screen sizes while maintaining the poster aesthetic.
- **Modern UI Patterns**: Uses subtle shadows, rounded corners, and engaging typography.

## Setup & Run
1. Ensure you have Flutter installed.
2. Clone or download this repository.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the application on your preferred device.

## Tech Stack
- Flutter
- Dart
